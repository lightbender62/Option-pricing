# Documentation

This directory contains the supporting documentation for the **Option Pricing** project.

## Directory Structure

```text
docs/
├── README.md
├── theory/
└── report/
```

### `theory/`

Contains the theoretical references and study material used during the implementation of the pricing models, including topics such as:

- Black-Scholes Theory
- Risk-Neutral Valuation
- Stochastic Processes
- Itô Calculus
- Option Greeks

### `report/`

Contains project reports and evaluation documents prepared during the development of the project, including the IITI Summer of Code (IITI-SOC) mid-evaluation report and future reports.

---

The documentation is intended to complement the source code by providing both the mathematical background and the project's development progress.
